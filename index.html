Hoi! Welkom op mijn portfolio website.

Mijn naam is Lars van Teeffelen en ik ben 16 jaar oud. Ik ben een Junior Software Developer op het Koning Willem I College in 's-Hertogenbosch. In 2020 ben ik hier begonnen en tot nu toe heb ik het altijd naar mijn zin gehad. Ik heb vele leuke activiteiten die ik in mijn vrije tijd doe. Ik werk bij de lokale supermarkt, speel videogames en ga graag naar vrienden. maar het leukste van alles is programmeren.

Ik ben in 2018 begonnen met het schrijven van code in de programmeertaal Java en heb daar kleine projecten mee gedaan. Later in 2019 toen ik wat meer van de taal begreep begon ik te kijken hoe ik plugins voor de videogame Minecraft kon maken, en heb ik daarin mijn programmeerkennis verder ontwikkeld door gesprekken te voeren met ervaren developers en samen te kijken naar mijn geschreven code voor verbeterpunten en meer efficiente manieren.

Ook ben ik door het maken van plugins voor Minecraft erachter gekomen hoe ik met externe APIs en met andere programmeurs efficient samen kan werken. Door software zoals Git en online services zoals Github te gebruiken voor makkelijk teamverband. Maar het meest belangrijke is dat ik ervaring opgedaan heb, gevonden wat ik leuk vind en wat ik later wil doen.
